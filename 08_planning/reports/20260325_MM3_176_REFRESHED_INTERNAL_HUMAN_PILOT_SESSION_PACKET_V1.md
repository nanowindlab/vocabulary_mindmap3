## Session Flow

### Flow 1. Search To Core

1. `돈` 검색
2. detail 진입
3. header / 핵심 / 번역 카드 읽기

### Flow 2. Relation Path

1. `관계` 탭 진입
2. 의미 관계어 또는 관련형 클릭
3. jump 이후 흐름 확인

### Flow 3. Expression Path

1. `표현` 탭 진입
2. `독립 항목 연결`과 `상세 연결 없음` 차이 확인
3. preview-only 표현의 의미 해석 확인

### Flow 4. Translation / Examples

1. 번역 언어 selector를 바꾼다
2. `요리하다`에서 main detail 반영 확인
3. `예문` 탭 helper와 source/order 이해 여부 확인

### Flow 5. Tree Semantics

1. `보다`로 `주제 및 상황 > 상황 미지정 > 일반 어휘` 해석 확인
2. `버리다`로 `분류 밖 항목` 해석 확인

### Flow 6. Performance Feel

1. category expansion
2. detail open/close
3. local delay 체감 여부 기록

### Fixed Performance Anchor

1. app first load -> first stable explorer render
2. `주제 및 상황` 탭 -> `역사` 계열 진입/확장 -> `5초` 관찰
3. `분류 밖 항목` 탭 진입 -> initial cluster 관찰
